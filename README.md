# DICOMCancerClassifier

A Python project for classifying medical images (DICOM) using machine learning.  
This prototype aims to support oncology decision-making by combining image analysis with clinical data.

## Features
- Preprocessing of DICOM medical images
- Training and evaluation of classification models
- Experiments with CNN architectures
- Visualization of predictions and metrics
